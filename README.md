## 프로젝트 개요
이 프로젝트는 **사용자가 직접 핸드폰 케이스를 디자인하고 제작할 수 있는 플랫폼**입니다.  
집에서 쉽고 간편하게 다양한 옵션을 선택하여 자신만의 핸드폰 케이스를 커스터마이징할 수 있습니다.  

---

## 실행 방법
1. 제공된 모든 이미지를 다운로드합니다.  
2. `PhoneCase.html` 파일을 실행하면 프로그램이 시작됩니다.  

---

## 구현 기능
- 핸드폰 케이스 적용 및 삭제  
- 케이스 색상 변경  
- 케이스 재질 변경  
- 케이스 뒷면에 캐릭터 이미지 삽입, 변경, 삭제
